# 📁 TagStripper v1.0

A lightweight, zero-dependency batch file utility built to instantly clean metadata tags, download traces, and media encoder string clutter from folders and file names.

## ✨ Features
- **Safe Cleanup:** Replaces system formatting symbols (`_`, `-`) with clean spacing properties without touching core metadata fields.
- **Smart Cleanup:** Aggressively filters tracker tags, bracket arrays (`[...]`, `(...)`), duplicate wording sequences, and standard web clutter strings.
- **Data Guard Integration:** Includes absolute extension lock protocols and collision overwrite protections.

## 🚀 How To Install
Do **not** download the source repository files for standard use. Instead:
1. Navigate to the **Releases** panel on the right sidebar of this page.
2. Download the standalone production asset: `TagStripper_Setup.exe`.
3. Launch the wizard setup installer. Upon completion, the operational documentation guide will automatically deploy inside your desktop browser workspace.

## 💻 Tech Stack
- **Language:** Python
- **GUI Engine:** Tkinter / Windows Native API Styling (Classic Lunar Framework Grid)
- **Deployment Build Array:** PyInstaller / Inno Setup Script Core
